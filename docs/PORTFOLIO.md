# Atomadic Portfolio Scope

Atomadic Lang is the verified intermediate-representation layer in the
Atomadic portfolio. It is not the public agent, the architectural enforcer, or
the network substrate. It is the compact, checkable form that AI-generated code
can be lowered into, tokenized, verified, transported, and eventually emitted by
models trained for the `.atm` grammar.

## Portfolio Map

| Project | Role | Relationship to Atomadic Lang |
|---|---|---|
| Atomadic | Sovereign AI operator and public autonomous presence | Uses Lang as a future substrate for compact, verifiable self-modification and memory-to-code workflows. |
| Atomadic Forge | Architectural substrate and MCP copilot for agents | Produces, audits, and organizes source code that can be lowered into `.atm` for verification and transport. |
| AAAA-Nexus | Coordination, memory, and multi-agent network layer | Can exchange verified capability artifacts and agent work products using Lang as a dense IR. |
| Atomadic Lang | Verified `.atm` IR and toolchain | Gives the ecosystem a machine-first, round-trippable representation for AI-emitted code. |

## Product Boundary

Atomadic Lang is deliberately scoped as a pre-alpha research toolchain:

- Python-to-`.atm` lowering for tier-organized packages.
- `.atm` parser, emitter, and byte-identical round-trip verification.
- Custom BPE training for `.atm` density experiments.
- Grammar/mask/latency benchmarking for edge-deployable verification.
- Documentation of the empirical claims, audit findings, and remaining work.

It does not yet claim:

- A production runtime for `.atm`.
- A trained model that natively emits `.atm`.
- Multi-language lowering beyond the current Python slice.
- Hardware-verified Raspberry Pi 5 latency numbers.

## Launch Positioning

Use Atomadic Lang as the research-facing proof that the Atomadic ecosystem is
not just prompting models to write code. The portfolio story is:

1. Atomadic decides what matters and acts in the world.
2. Forge keeps AI-generated code architecturally coherent.
3. AAAA-Nexus coordinates agents, memory, and work.
4. Lang compresses and verifies the code artifacts those systems produce.

That makes Lang the portfolio's "verified IR for sovereign agents": a compact
machine form for software that was planned, generated, reviewed, and evolved by
AI systems.
